## Instrucciones de Ejecución

- tener instalado `c++ Builder IDE`o `RAD Studio`

- Clona este repositorio en tu máquina local.

  ```bash
  git clone https://github.com/sebanovo/files-project.git
  ```

- abrir el archivo `.cbproj` con tu IDE

- Compila y Listo ✨.
